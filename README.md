# Currency Converter in Python
* currency.py - module that contains all the functions.
* textcurrency.py - unit test that test all the functions in currency.py.
* exchangeit.py - Entry point. Run this script "python exchangeit.py".

## python exchangeit.py
3-letter code for original currency: USD  
3-letter code for the new currency: EUR  
Amount of the original currency: 100.00  
You can exchange 100.0 USD for 88.641 EUR.  

Uses Cornell University Python modules and web APIs.
